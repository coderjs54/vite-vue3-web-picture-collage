# 拼图工坊

一个基于 Vue 3 构建的网页端图片拼图工具，支持自由画布模式，用户可以上传图片并在画布上自由拖拽、缩放、旋转，创造个性化的拼图作品。

## 功能特性

- **图片上传**：支持多选图片上传，图片自动添加到底部图片库
- **自由画布**：支持自定义画布尺寸（宽 400-1200px，高 300-900px）
- **拖拽定位**：图片按下即跟随鼠标移动，操作流畅直观
- **缩放调整**：支持8个控制点（4个角+4个边）精确调整图片大小
- **旋转功能**：支持360°自由旋转，带有角度控制点
- **文字添加**：支持在画布上添加文字，可自定义颜色和大小
- **撤销/重做**：完整的操作历史记录，支持无限撤销重做
- **重置功能**：一键恢复图片到初始状态
- **层级管理**：自动管理图层顺序，选中图片自动提升层级
- **拼图下载**：支持将作品导出为 PNG 格式保存到本地

## 技术栈

- **框架**：Vue 3 + TypeScript
- **构建工具**：Vite 8
- **样式**：SCSS
- **图标**：SVG 矢量图标

## 使用说明

### 上传图片

1. 点击顶部工具栏的「上传图片」按钮
2. 在弹出的文件选择对话框中选择一张或多张图片
3. 图片会自动添加到底部图片库中

### 添加图片到画布

- **拖拽方式**：从底部图片库拖拽图片到画布任意位置
- **点击方式**：直接点击底部图片库中的图片，图片会自动添加到画布中心

### 编辑图片

1. 点击画布上的图片选中它
2. **移动**：拖动图片到目标位置
3. **缩放**：拖动角控制点调整宽高，拖动边控制点单独调整宽度或高度
4. **旋转**：拖动顶部旋转控制点进行旋转
5. **重置**：点击图片上方的重置按钮恢复到初始状态
6. **删除**：点击图片上方的删除按钮移除图片

### 添加文字

1. 点击顶部工具栏的「添加文字」按钮
2. 在弹出的输入框中输入文字内容
3. 按 Enter 键确认添加
4. 文字会出现在画布中心，支持缩放、旋转和删除操作

### 调整画布

- 修改顶部工具栏中的画布尺寸输入框
- 点击颜色选择器调整画布背景色
- 使用缩放按钮调整画布显示比例

### 下载拼图

点击顶部工具栏的「下载拼图」按钮，当前画布状态会自动保存为 PNG 格式图片。
